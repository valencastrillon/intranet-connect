# Intranet Empresarial

Aplicación web tipo intranet diseñada para mejorar la comunicación, gestión documental y acceso a la información dentro de una organización.

## Descripción

Esta intranet permite a los colaboradores interactuar en un entorno centralizado donde pueden compartir información, consultar documentos y mantenerse alineados con los objetivos de la empresa.

Incluye funcionalidades sociales tipo feed, gestión de documentos institucionales y personalización de usuario.

## Tecnologías

- Vite
- React
- TypeScript

## Funcionalidades principales

### Autenticación
- Inicio de sesión
- Cierre de sesión
- Actualización de nombre de usuario
- Cambio de contraseña mediante correo electrónico

### Feed colaborativo
- Publicaciones tipo red social
- Comentarios entre colaboradores
- Adjuntar imágenes en publicaciones

### Gestión documental
- Manuales
- Procedimientos
- Actas
- Subida de documentos
- Filtros:
  - Favoritos
  - Más recientes a más antiguos

### Información institucional
- Misión
- Visión
- Objetivos
- Organigrama

### Glosario
- Glosario técnico
- Glosario institucional

### UI/UX
- Modo oscuro
- Diseño responsive
